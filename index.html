☁️ GitHub 동기화

내 거래 기록을 GitHub 저장소의 JSON 파일로 올려, 다른 기기·브라우저에서도 같은 데이터를 쓸 수 있어요. 설정은 이 브라우저에만 저장됩니다.

github.com → Settings → Developer settings → Personal access tokens. Fine-grained 토큰 권장 — 대상 저장소에 Contents: Read & write 권한. 토큰은 이 기기에만 저장되며 어디에도 전송되지 않아요(GitHub 직접 호출).
⚠️ 공개 저장소에 올리면 누구나 볼 수 있어요. 비공개(private) 저장소를 권장합니다 — 없으면 아래 버튼으로 만들 수 있어요.
